# Data Jobs Dashboard | Power BI Project

![Data Jobs Dashboard Overview](../images/data-jobs-v1-overview.png)

## 📊 Introduction

This Power BI dashboard analyzes the data job market using a real-world dataset of 2024 data-related job postings. The dataset includes details such as job titles, salaries, locations, work-from-home availability, job platforms, and other job market information.

I created this dashboard while following the **Power BI for Data Analytics - Full Course for Beginners** by **Luke Barousse**. Along with the course structure, I customized the dashboard design, improved the user experience, and added my own analysis to make the report more useful for job seekers and career switchers.

The main purpose of this dashboard is to help users quickly understand job demand, salary trends, remote work options, and job distribution across different roles and locations.

---

## 🎯 Project Objective

The objective of this project is to answer key job-market questions such as:

- Which data roles have the highest number of job postings?
- What are the median yearly and hourly salaries for different data roles?
- Which jobs offer work-from-home opportunities?
- Which platforms have the most data job postings?
- How are data jobs distributed across different countries?
- How can users drill into a specific job title for deeper analysis?

## 📌 Dashboard Overview

The report is divided into two main pages:

---

## Page 1: High-Level Job Market Overview


![Dashboard Page 1](/images/Project1_Page1.png)


This page provides a quick summary of the overall data job market.

### This page includes:

- Total job count
- Median yearly salary
- Median hourly salary
- Top data job titles
- Job count by location
- Salary comparison by role
- Work-from-home availability
- Job title slicer for filtering the dashboard

This page works as the main overview where users can quickly understand the size of the job market and compare major data roles.

---

## Page 2: Job Title Drill-Through Analysis

![Dashboard Page 2](/images/Project1_Page2.png) 

This page provides a deeper analysis for a selected job title.

From the main dashboard, users can drill through into a specific role and view more detailed information.

### This page includes:

- Salary details for the selected job title
- Work-from-home statistics
- Job posting platforms
- Location-based job distribution
- Global map view of job opportunities
- Detailed breakdown of the selected role

This drill-through page makes the dashboard more interactive and useful because users can move from a high-level summary to a focused role-specific analysis.

## ✨ My Custom Additions and Improvements

Along with following the course dashboard, I made my own improvements to make the report more polished and user-friendly.

### 1. Improved Dashboard Layout

I customized the layout to make the dashboard easier to read and navigate. I focused on spacing, alignment, visual hierarchy, and clean formatting so that the most important information is visible quickly.

### 2. Enhanced Job Market Storytelling

Instead of only showing charts, I organized the visuals around practical questions a job seeker may have, such as salary expectations, remote work availability, job locations, and role-specific details.

### 3. Drill-Through User Experience

I used drill-through functionality to allow users to move from the main dashboard to a detailed job-title page. This creates a better analytical flow because users can first view the overall market and then explore one specific role in detail.

### 4. Salary and Remote Work Focus

I emphasized salary and work-from-home insights because these are important factors for students, job seekers, and career switchers when comparing roles.

### 5. Cleaner Visual Design

I adjusted visual formatting, chart placement, labels, and dashboard structure to make the report more professional and portfolio-ready.
