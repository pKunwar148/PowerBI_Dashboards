# Data Jobs Market Analysis | Power BI Project

## Project Overview

This repository contains two versions of a Power BI dashboard project focused on analyzing the data job market. The project uses a real-world dataset of data-related job postings to explore job demand, salary patterns, remote work availability, company hiring activity, skill requirements, and role-based market trends.

The project was created as part of my hands-on Power BI learning journey and was expanded with additional analysis, improved dashboard design, interactive filters, and deeper insights to make it more useful for job seekers, students, and career switchers.

## Dashboard Versions

### Version 1: Market Overview Dashboard

The first version focuses on a high-level overview of the data job market. It includes job count, salary comparison, work-from-home availability, job distribution by location, platform-based job postings, and drill-through analysis by job title.

This version helped build the foundation for dashboard design, Power Query transformation, KPI cards, slicers, basic DAX measures, and role-specific drill-through analysis.

Folder:

```text
v1-market-overview/
```

### Version 2: Advanced Market Analysis Dashboard

The second version expands the project with deeper analysis and more interactive reporting features. It includes salary analysis, company-level hiring trends, job schedule type analysis, skill demand analysis, skill vs salary comparison, interactive category and measure analysis, and estimated take-home pay analysis.

This is the advanced version of the project and represents the main portfolio-ready dashboard.

Folder:

```text
v2-advanced-analysis/
```

## Key Improvements from V1 to V2

* Expanded the dashboard from a basic market overview into a multi-page analytical report
* Added salary analysis across job titles and pay types
* Added company-level hiring and compensation comparison
* Added job schedule type analysis, including full-time, contract, internship, and part-time roles
* Added skill demand analysis to identify frequently requested data skills
* Added skill vs salary analysis to explore how skill requirements relate to compensation
* Added interactive category and measure selection using parameters
* Added estimated take-home pay analysis using a user-defined deduction rate
* Improved dashboard layout, storytelling, navigation, and overall user experience

## Key Insights

* Data Analyst, Data Engineer, and Data Scientist roles show strong demand in the dataset.
* Python and SQL appear as two of the most frequently requested skills in data-related jobs.
* Senior and specialized technical roles generally show higher median salaries.
* Full-time roles dominate the data job market compared to contract, internship, and part-time roles.
* Companies with the highest number of job postings do not always offer the highest median salaries.
* Roles requiring more skills often show higher salary potential, especially senior and engineering-focused roles.

## How to View the Dashboards

1. Open the desired version folder.
2. Download the `.pbix` file.
3. Open the file using Power BI Desktop.
4. Use slicers, filters, and navigation options to explore the dashboard.
5. Review each dashboard page to understand the job market insights.
